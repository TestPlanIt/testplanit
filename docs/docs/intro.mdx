# Welcome to TestPlanIt Documentation

TestPlanIt is an open-source tool designed to streamline your testing process by providing a central platform for creating, managing, and executing both manual and automated test plans.

Whether you're managing complex test suites, coordinating exploratory testing sessions, or simply looking for a better way to organize your test cases, TestPlanIt aims to help.

## What's in this documentation?

This documentation will guide you through setting up and using TestPlanIt effectively. It covers:

- **Installation:** Setting up TestPlanIt for local development or deployment.
- **Getting Started:** A quick tour of the user interface and core concepts.
- **Guides:** Detailed explanations of key features like test case management, test execution, session tracking, and reporting.
- **(Coming Soon) API Reference:** Information on interacting with TestPlanIt programmatically.

## Ready to dive in?

- If you haven't set up TestPlanIt yet, head to the **[Installation Guide](./installation)**.
- To get a feel for the application, check out the **[Getting Started](./getting-started)** tutorial.
